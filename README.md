# Leads Tracking App

![Leads Tracking App Icon](https://github.com/shubham-pawar/leads_tracking_app/blob/main/icon.png)

A simple and modern **Progressive Web App (PWA)** for tracking leads and saving important links in one place. It is built with web technologies and provides an app-like experience that can be installed, used offline, and accessed across multiple devices.

## Features
- Save and manage leads easily
- Add links from the browser
- Delete all saved leads when needed
- Works as a Progressive Web App
- Can be installed on desktop or mobile
- Supports offline use

## What is a PWA?
A **Progressive Web App (PWA)** is a web application built using standard web technologies that behaves like a native app.  
It can run on multiple platforms from a single codebase, be installed on a device, work offline, and integrate with the operating system and other apps.

## How It Works
1. Open the app in your browser.
2. Save leads manually or from the current tab.
3. View your saved links anytime.
4. Use the app like a regular installable application.

## Tech Stack
- HTML
- CSS
- JavaScript
- Progressive Web App features

## Installation
1. Clone the repository.
2. Open the project in your browser or host it locally.
3. Install the app from the browser if prompted.
